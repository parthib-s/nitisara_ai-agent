# Captain AI Firebase Demo

Conversational agent (Captain) for order management, trade compliance, and rate estimation.
- Python Flask backend
- LangChain with Gemini API
- Firebase (Realtime DB)
- Minimal HTML demo frontend

## Setup

1. Add your Firebase admin JSON key to the repo.
2. Fill `.env` with your keys and Firebase project info.
3. `pip install -r requirements.txt`
4. In one terminal: `cd backend && python main.py`
5. Open `frontend/index.html` in your browser.
